## How to get on this list

1. Open an issue or pick up an existing one.
2. Send a pull request following the guidelines in [`README.md`](./README.md#contributing--development-notes) — run `pnpm run lint` and `pnpm test` before pushing.
3. Once your PR is merged, add yourself to the **Contributors** section above (or a maintainer will).

A full, automatically-generated list of everyone who has committed to the repository is also available on GitHub:
<https://github.com/GitFarhanS/cryptoDraw/graphs/contributors>
